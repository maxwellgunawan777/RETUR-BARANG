STOP! Database Belum Ada

Aplikasi Anda sudah benar, tapi "gudang penyimpanan" (Firestore) di Google belum dibuat. Aplikasi tidak akan jalan sampai Anda membuatnya.

Langkah Pembuatan (Wajib):

  1. Klik Link Ini (Firebase Console).
  2. Pilih proyek aplikasi-retur-barang.
  3. Di menu kiri, cari & klik Firestore Database (atau Cloud Firestore).
  4. Klik tombol Create Database.
  5. Lokasi? Biarkan default, klik Next.
  6. PENTING: Pilih Start in test mode. Lalu klik Create.

Izin Akses Ditolak

Database sudah ada, tapi menolak izin tulis. Mohon update "Rules".

1. Copy Kode Ini:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }

(Klik kode untuk menyalin)

2. Paste di Sini:

  1. Buka Firebase Console > Firestore Database.
  2. Klik tab Rules.
  3. Hapus semua kode lama, Paste kode baru, klik Publish.
