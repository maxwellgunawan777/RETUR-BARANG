Database Belum Dibuat!

Error ini muncul karena Anda belum membuat Database Firestore di Firebase Console. Aplikasi tidak bisa menyimpan data tanpanya.

Langkah Perbaikan (Wajib):

  1. Buka Firebase Console Anda.
  2. Klik pada proyek aplikasi-retur-barang.
  3. Di menu sebelah kiri, klik Build lalu pilih Firestore Database.
  4. Klik tombol Create Database.
  5. Pilih lokasi (biarkan default), klik Next.
  6. Pilih Start in test mode (PENTING!), lalu klik Create.

Izin Akses Ditolak

Database sudah ada, tapi menolak izin tulis. Anda perlu memperbarui "Rules".

1. Copy Kode Ini:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }

(Klik kode di atas untuk menyalin)

2. Paste di Sini:

  1. Buka Firebase Console > Firestore Database.
  2. Klik tab Rules.
  3. Hapus kode lama, Paste kode baru, lalu klik Publish.
